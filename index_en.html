Simulation of a Radio Communication System

ENSEIRB–MATMECA: Second-year project

Project Objectives

This project introduced radio link simulation using Keysight ADS. The goal was to design a QPSK transmission chain including amplifiers, mixers, filters, oscillators, and antennas.

Key Concepts

  • QPSK modulation and demodulation.
  • Raised cosine filtering.
  • RF transmission/reception chain.
  • Amplifier non-linearity and P1dB compression point.
  • Propagation models and path loss.
  • Constellation and eye diagram analysis.
  • Bit Error Rate (BER) evaluation.
